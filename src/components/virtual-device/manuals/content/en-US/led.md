Use this device to observe one logic output from the design.

## Good for

- Heartbeat indicators
- State bits, handshake signals, and reset-done flags
- Quickly confirming whether an output is really toggling

## Notes

- This device only samples and displays. It never drives signals back into the design.
- If the real-time stream is not running, the lamp state reflects only the latest snapshot, not a continuous on-board waveform.
